# Autism Detection Research Exploration

## Overview
This repository contains exploratory work from a research project focused on early autism detection using facial image datasets. I worked on this project as a research apprentice under Dr. Haohan Wang at the University of Illinois Urbana-Champaign.

The goal of the project was to explore how machine learning techniques could potentially analyze facial images and identify patterns that may help with early autism detection.

---

## What I Worked On
During the project I focused mainly on the data side of the research process. My work involved:

- exploring publicly available datasets from platforms such as Kaggle, UCI, and OpenNeuro
- reviewing and organizing image data so it could be used for analysis
- understanding how datasets are prepared before training machine learning models
- observing the overall workflow used in machine learning research projects

This repository contains a Python script used while exploring dataset preparation and basic model experimentation.
